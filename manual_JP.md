# Dante プリセット XML → Excel 変換ツール マニュアル

## ■ 概要
Dante Controller で書き出したプリセットファイル（XML）を Excel ファイルに変換するツールです。
デバイス一覧、パッチマトリクス、フロー情報などを自動で生成します。
AES67 の設定情報にも対応しています。

## ■ 必要な環境
- Windows 10 / 11
- Microsoft Excel がインストールされていること
- 追加ソフトのインストールは不要です

## ■ ファイル構成
- `DanteToExcel.ps1`: 変換ツール本体
- `DanteToExcel.bat`: 実行用バッチファイル

## ■ 使い方
1. `DanteToExcel.ps1` を任意のフォルダにコピーします。
2. 同じフォルダに、変換したい Dante プリセット XML ファイルを置きます。
3. `DanteToExcel.ps1` を右クリックし「PowerShell で実行」を選択するか、`DanteToExcel.bat` をダブルクリックします。
4. メニューが表示されるので、モードを選択します（通常は `1` を選択）。
5. 処理が完了すると、同じフォルダに Excel ファイルが生成されます。

## ■ 出力モード
- **Default (1)**: 重要な項目に絞った概要データ。
- **Detail (2)**: チャンネルごとの詳細やサブスクリプション情報など、すべてのデータ。

## ■ 注意事項
- 変換中は Excel がバックグラウンドで起動します。処理が完了するまで Excel を手動で操作しないでください。
- チャンネル数が多い場合、パッチマトリクスの生成に時間がかかることがあります。
